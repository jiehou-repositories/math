The Set of Real Numbers

Definition

  1. A set \(\mathbb{R}\).
  2. Operations:
    • Addition \(+:\ \mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\)
      \(\forall\ a, b \in \mathbb{R}\), \(+(a, b) = a + b\) is called the sum of \(a\) and \(b\).
    • Multiplication \(\cdot:\ \mathbb{R} \times \mathbb{R} \rightarrow \mathbb{R}\)
      \(\forall\ a, b \in \mathbb{R}\), \(\cdot(a, b) = a \cdot b = ab\) is called the product of \(a\) and \(b\).
  3. Relation:
    Order \(\leq:\ \mathbb{R} \times \mathbb{R} \rightarrow \{\text{True}, \text{False}\}\).
  4. Conditions:
    1. Axioms for addition
      • Additive commutativity: \(\forall\ a, b \in \mathbb{R}\), \(a + b = b + a\).
      • Additive associativity: \(\forall\ a, b, c \in \mathbb{R}\), \(a + (b + c) = (a + b) + c\).
      • Additive identity element: \(\exists\ 0 \in \mathbb{R}\), \(\forall\ a \in \mathbb{R}\), \(a + 0 = a\).
      • Additive inverse element: \(\forall\ a \in \mathbb{R}\), \(\exists\ b \in \mathbb{R}\), \(a + b = 0\).
        \(b\) is denoted as \(-a\).
    2. Axioms for multiplication
      • Multiplicative commutativity: \(\forall\ a, b \in \mathbb{R}\), \(a \cdot b = b \cdot a\).
      • Multiplicative associativity: \(\forall\ a, b, c \in \mathbb{R}\), \(a \cdot (b \cdot c) = (a \cdot b) \cdot c\).
      • Multiplicative identity element: \(\exists\ 1 \in \mathbb{R} \backslash \{0\}\), \(\forall\ a \in \mathbb{R}\), \(a \cdot 1 = a\).
      • Multiplicative inverse element: \(\forall\ a \in \mathbb{R} \backslash \{0\}\), \(\exists\ b \in \mathbb{R}\), \(a \cdot b = 1\).
        \(b\) is denoted as \(a^{-1}\).
    3. Distributive property: \(\forall\ a, b, c \in \mathbb{R}\), \((a + b) \cdot c = a \cdot c + b \cdot c\).
    4. Order axioms
      • \(\forall\ a \in \mathbb{R}\), \(a \leq a\).
      • \(\forall\ a, b \in \mathbb{R}\), \((a \leq b) \land (b \leq a) \Rightarrow (a = b)\).
      • \(\forall\ a, b, c \in \mathbb{R}\), \((a \leq b) \land (b \leq c) \Rightarrow (a \leq c)\).
      • \(\forall\ a, b \in \mathbb{R}\), \((a \leq b) \lor (b \leq a)\).
    5. \(\forall\ a, b, c \in \mathbb{R}\), \((a \leq b) \Rightarrow (a + c \leq b + c)\).
    6. \(\forall\ a, b \in \mathbb{R}\), \((0 \leq x) \land (0 \leq y) \Rightarrow (0 \leq x \cdot y)\).
    7. Continuity (Axiom of completeness): \(\forall\ A, B \subseteq \mathbb{R}\), if \(\forall\ a \in A\) and \(\forall\ b \in B\), there is \(a \leq b\), then \(\exists\ c \in \mathbb{R}\) such that \(\forall\ a \in A\) and \(\forall\ b \in B\), \(a \leq c \leq b\).
