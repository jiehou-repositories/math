Vector Spaces

Vector spaces

  • A set \(V\).
  • Functions:
    • Addition \(+:\ V \times V \rightarrow V\)
      \(\forall\ \textbf{u}, \textbf{v} \in V\), \(+(\textbf{u}, \textbf{v}) = \textbf{u} + \textbf{v}\).
    • Scalar multiplication \(\cdot:\ F \times V \rightarrow V\)
      \(\forall\ a \in F\) and \(\forall\ \textbf{v} \in V\), \(\cdot(a, \textbf{v}) = a\cdot\textbf{v} = a\textbf{v}\).
  • Properties:
    • Commutativity:
      \(\forall\ \textbf{u}, \textbf{v} \in V,\ \textbf{u} + \textbf{v} = \textbf{v} + \textbf{u}\).
    • Associativity:
      \(\forall\ \textbf{u}, \textbf{v}, \textbf{w} \in V\), \((\textbf{u} + \textbf{v}) + \textbf{w} = \textbf{u} + (\textbf{v} + \textbf{w})\)
      \(\forall\ \textbf{v} \in V\) and \(\forall\ a, b \in F\), \((a\cdot_{F} b)\textbf{v} = a(b\textbf{v})\).
    • Additive identity:
      \(\exists\ \textbf{0} \in V\), \(\forall\ \textbf{v} \in V\), \(\textbf{v} + \textbf{0} = \textbf{v}\).
    • Additive inverse:
      \(\forall\ \textbf{v} \in V\), \(\exists\ \textbf{u} \in V\), \(\textbf{v} + \textbf{u} = \textbf{0}\).
      \(\textbf{u} = -\textbf{v}\).
    • Multiplicative identity:
      \(\forall\ \textbf{v} \in V\), \(1\cdot \textbf{v} = \textbf{v}\).
    • Distributive properties:
      \(\forall\ a \in F\) and \(\forall\ \textbf{u}, \textbf{v} \in V\), \(a(\textbf{u} + \textbf{v}) = a\textbf{u} + a\textbf{v}\).
      \(\forall\ a, b \in F\) and \(\forall\ \textbf{v} \in V\), \((a +_{F} b)\textbf{v} = a\textbf{v} + b\textbf{v}\).

Subspaces

A subset \(U\) of a vector space \(V\) is called a subspace of \(V\) if \(U\) is also a vector space.

Linear combination

\(\forall\ n \in \mathbb{Z}^{+}\), \(\forall\ a_{1}, \dots, a_{n} \in F\), \(\forall\ \textbf{v}_{1}, \dots, \textbf{v}_{n} \in V\), a linear combination of \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) is a vector of the form \(a_{1}\textbf{v}_{1} + \cdots + a_{n}\textbf{v}_{n}\).

Span

\(\forall\ n \in \mathbb{Z}^{+}\), \(\forall\ \textbf{v}_{1}, \dots, \textbf{v}_{n} \in V\), the set of all linear combinations of \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) is called the span of \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\), denoted \(\text{span}(\textbf{v}_{1}, \dots, \textbf{v}_{n})\). In other words, \[ \text{span}(\textbf{v}_{1}, \dots, \textbf{v}_{n}) = \{a_{1}\textbf{v}_{1} + \cdots + a_{n}\textbf{v}_{n}\ |\ n \in \mathbb{Z}^{+},\ a_{1}, \dots, a_{n} \in F,\ \textbf{v}_{1}, \dots, \textbf{v}_{n} \in V\}. \] The span of \(()\) is defined to be \(\{\textbf{0}\}\).
If \(\text{span}(\textbf{v}_{1}, \dots, \textbf{v}_{n}) = V\), then we say that \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) spans \(V\).
  • A vector space is called finite-dimensional if there exist some vectors in it that span the space.
  • A vector space is called infinite-dimensional if it is not finite-dimensional.

Linearly independent and linearly dependent

\(\forall\ n \in \mathbb{Z}^{+}\), \(\forall\ \textbf{v}_{1}, \dots, \textbf{v}_{n} \in V\), \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) is called linearly independent if the only choice of \(a_{1}, \dots, a_{n} \in F\) that makes \(a_{1}\textbf{v}_{1} + \cdots + a_{n}\textbf{v}_{n}\) equal \(\textbf{0}\) is \(a_{1} = \cdots = a_{n} = 0\).
\(\forall\ n \in \mathbb{Z}^{+}\), \(\forall\ \textbf{v}_{1}, \dots, \textbf{v}_{n} \in V\), \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) is called linearly independent if it is not linearly independent. In other words, \((\textbf{v}_{1}, \dots, \textbf{v}_{n})\) is linearly dependent if \(\exists\ a_{1}, \dots, a_{n} \in F\), not all \(0\), such that \(a_{1}\textbf{v}_{1} + \cdots + a_{n}\textbf{v}_{n}\).
